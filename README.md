# ActionsBoard-Solo

コンテキスト指向型 GitHub Actions 監視 Chrome 拡張機能

## 概要

ActionsBoard-Solo は、GitHub Actions の実行状態と GitHub Pages のデプロイステータスをリアルタイムで追跡し、開発者の「待ち時間」を最小化するための Chrome 拡張機能です。
特に複数リポジトリを跨ぐプロジェクトにおいて、自分に関連するワークフローや共有ブランチの状態を効率的に監視することを目的としています。

## 特徴

- **コンテキスト指向型ダッシュボード:** ユーザーの役割（開発者、チームリード、運用保守）に合わせた表示モードの切り替え。
- **2段階デプロイ追跡:** Actions の完了から GitHub Pages への反映までを一気通貫で監視。
- **ノイズレス通知:** 自分に関連する重要な変更のみを通知し、認知負荷を軽減。
- **ローカル完結型:** 外部サーバーを介さず、ブラウザから直接 GitHub API を使用。
- **Material Design 3:** モダンで直感的なユーザーインターフェース。

## ディレクトリ構成

- `projects/app/`: 拡張機能の本体コード
